This page tracks notable product changes for Chronote users.

## 2026

### Documentation platform launch

- Public product docs moved to `docs.chronote.gg`.
- Docs now run in the main Chronote monorepo with CI checks and deployment.
- User-facing PRs now include docs updates or use a `docs-exempt` path.
