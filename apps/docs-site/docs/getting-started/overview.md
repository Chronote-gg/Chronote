Chronote records Discord voice meetings, transcribes the conversation, and posts structured notes back to your channel.

## What you need

- A Discord server where you can add bots.
- Permission to create or manage voice and text channels.
- A Chronote subscription tier that matches your team usage.

## First setup checklist

1. Add Chronote to your server.
2. Run `/onboard` for guided setup.
3. Configure context with `/context`.
4. Add project terms with `/dictionary`.
5. Start with `/startmeeting` or configure `/autorecord`.

## What happens during a meeting

- Chronote captures voice and chat activity.
- Audio is transcribed and cleaned for clarity.
- A notes summary is generated and posted in Discord.
- Meeting history is saved for later access in Chronote surfaces.
