This page covers practical admin tasks for server owners and operators.

## Setup sequence

1. Add Chronote to the server and confirm command registration.
2. Configure context and dictionary defaults.
3. Enable auto-record where your team needs automation.
4. Confirm billing tier and payment status.

## Access model

- Meeting notes in Discord follow channel permissions.
- Portal views reuse Discord channel permissions with attendee exceptions.
- Sensitive operations require elevated permissions or meeting ownership.

## Operational tips

- Use dedicated meeting channels for cleaner notes and audits.
- Keep context concise and update it when project terms change.
- Review note corrections with the meeting owner when possible.
