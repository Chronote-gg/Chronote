Chronote ships core meeting capture features plus quality and governance controls.

## Core features

- Manual meeting start and end controls.
- Auto-record at server or channel scope.
- AI-generated notes from meeting audio.
- Meeting history storage with versioned notes.

## Quality controls

- Prompt context at server, channel, and meeting level.
- Dictionary terms injected into transcription and summary prompts.
- Minimal-edit correction flow with review before apply.

## Admin and billing

- Guild-scoped billing with Stripe checkout and portal.
- Access checks for notes and meeting surfaces.
